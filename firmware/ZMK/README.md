# ZMK Firmware

ZMK is a modern, wireless-first firmware for mechanical keyboards, built on the Zephyr RTOS. It provides excellent battery life and advanced features for PNCATEHO variants.

## Configuration

Different PNCATEHO variants require different ZMK configurations:

### nizkoteno

For nizkoteno, use this dedicated configuration:

- [ZMK Config for nizkoteno](https://github.com/inpudiy/zmk-PNCATEHO_Welcome_keyboard)

### Other Variants

For all other PNCATEHO variants:

- [ZMK Config for PNCATEHO](https://github.com/aroum/zmk-PNCATEHO)

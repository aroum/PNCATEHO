# nizkore Firmware

nizkore is a powerful sampler firmware for PNCATEHO, enabling playback of audio samples and real-time control over playback parameters. It's a port of the pikocore sampler, optimized for the PNCATEHO form factor and 2-key chord interface.

## Overview

This firmware transforms your PNCATEHO into a portable audio sampler with intuitive 2-key chord-based control. Originally developed for nizkore, it can be adapted to other PNCATEHO variants using RP2040-based microcontrollers.

## Repository

- [nizkore](https://github.com/aroum/nizkore/)

## Compatibility

- **Supported:** nizkore and other PNCATEHO variants with RP2040-based microcontrollers
- **Unsupported:** dozateno (different microcontroller architecture)

For more information about pikocore, visit the [original project](https://github.com/schollz/pikocore).

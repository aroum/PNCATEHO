# MIDI Firmware

This firmware transforms your PNCATEHO into a MIDI controller, enabling seamless integration with Digital Audio Workstations (DAWs), synthesizers, and other MIDI-compatible devices.

## Overview

A straightforward MIDI keyboard implementation originally developed for nizkore. The firmware can be adapted to other PNCATEHO variants by reconfiguring pin assignments.

## Repository

- [midi-nizkoteno](https://github.com/aroum/midi-nizkoteno)

## MIDI Channel Assignment

Each key combination can be configured to transmit on different MIDI channels. Refer to the source code or firmware documentation for detailed channel mapping.

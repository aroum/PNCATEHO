# pra32-u Firmware

pra32-u is a compact synthesizer firmware for nizkoteno, providing real-time synthesis and sound generation capabilities. This is a port of the PRA32-U synthesizer, optimized for the PNCATEHO keyboard and 2-key chord control scheme.

## Overview

Transform your PNCATEHO into a portable synthesizer. Originally implemented for nizkore, this firmware can be adapted to other PNCATEHO variants equipped with RP2040-based microcontrollers.

## Repository

- [pra32-u-pico-sdk](https://github.com/aroum/pra32-u-pico-sdk/)

## Compatibility

- **Supported:** nizkore and other PNCATEHO variants with RP2040-based microcontrollers
- **Unsupported:** dozateno

Refer to the repository's documentation for a complete chord-to-function mapping.

For more information about PRA32-U, visit the [original project](https://github.com/risgk/digital-synth-pra32-u).

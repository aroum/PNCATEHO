# RMK Firmware

RMK (Rust Mechanical Keyboard) is a high-performance, modern firmware written in Rust for RP2040-based microcontrollers. It offers excellent performance and a clean, maintainable codebase.

## Overview

This is the primary RMK implementation for PNCATEHO, originally developed for nizkoteno. However, it can be adapted to other PNCATEHO variants by adjusting the pin configuration in the source code.

## Repository

- [rmk-nizkoteno](https://github.com/aroum/rmk-nizkoteno/)

## Compatibility

- **Supported:** nizkoteno and other PNCATEHO variants with RP2040-based microcontrollers
- **Unsupported:** dozateno

For more information, visit the [RMK project](https://github.com/HaoboGu/rmk).
